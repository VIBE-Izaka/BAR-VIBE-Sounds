# BAR-VIBE-Sounds

**Author:** VIBE  
**Game:** Beyond All Reason (BAR)  
**File:** forcestart_vote_sound.lua  
**Type:** Notification / Audio Widget  

## Overview

VIBE-Sounds is a lightweight UI widget that enhances the voting experience in Beyond All Reason by playing a custom notification sound whenever a **ForceStart vote is initiated** in the lobby or pre-game.

The widget listens for vote start messages and provides an immediate audio cue so players don’t miss important vote activity, even in busy lobbies.

## Features

- Plays a custom sound when a **ForceStart vote starts**  
- Works in lobby and pre-game environments  
- Designed for future expansion into a full notification system  

## Installation

1. Download the widget files  
2. Copy them into your BAR folder:

`Beyond-All-Reason\data\LuaUI\`
